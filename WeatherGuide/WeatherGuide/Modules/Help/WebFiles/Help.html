WeatherGuide Help Center

Some cool features:

  1. Add and save locations: You can add a location by dropping a pin on the map by a simple tap.
  2. Edit locations: Added locations can be deleted by clicking pencil icon in the home page.
  3. View current weather conditions: Current forecast can we viewed on selecting any added location.
